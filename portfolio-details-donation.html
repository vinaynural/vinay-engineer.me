Project Overview

Developed a comprehensive donation management platform for Thaagam Foundation, a prominent nonprofit organization serving over 200,000 followers. This enterprise-grade solution manages the complete donation lifecycle from collection to distribution, featuring multiple specialized dashboards for donors, administrators, field officers, and finance teams.

The platform has revolutionized contributions, bringing transparency, efficiency, and accountability. With integrated AI-powered fraud detection and multi-channel communication capabilities, the system ensures genuine donations reach beneficiaries.

Key Features

  • Multi-Dashboard Architecture: Four specialized interfaces for Donors, Admins, Field Officers, and Finance.
  • AI-Powered Fraud Detection: Gemini API integration for donation validation.
  • Multi-Channel OTP Verification: Secure donor verification via WhatsApp, Email, and SMS.
  • Real-time Tracking: Live donation tracking with photo evidence.
  • Automated Receipt Generation: Instant 80G tax exemption certificates.

Technical Implementation

Backend: Django with REST Framework.

Database: PostgreSQL with optimized schema and partitioning.

Cloud & AI: AWS (EC2, SES, SNS, S3) and Gemini API.

Project Impact

  • Processing 2,000+ donations monthly (₹50L+ value)
  • Reduced administrative overhead by 60%
  • Prevented ₹2L+ in fraudulent donations via AI
  • Improved donor satisfaction score to 9.1/10
